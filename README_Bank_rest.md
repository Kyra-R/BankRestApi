# Система управления банковскими картами

# Bank REST API

## 1. Описание проекта
**Bank REST API** — это приложение с возможностью регистрации для управления пользователями и картами.
 
Некоторые возможности пользователей:
- Перевод с карты на карту
- Запрос на блокировку карты
- Просмотр своих карт
 
Некоторые возможности администраторов:
- Работа с пользователями, в т.ч. просмотр, создание, удаление, изменение параметров
- Блокировка карт по запросу
- Создание, удаление, изменение и просмотр карт

Данные номера карты маскируются для пользователя, но не для админа. Пароли и номера карт в базе данных зашифрованы.

---

## 2. Требования и пояснения для самостоятельной сборки jar

- Java 17 
- Maven 

В .env и pom.xml находятся некоторые данные (такие как параметры БД), которые можно менять на свое усмотрение.

---

## 3. Запуск
Скачайте архив из репозитория и распакуйте файлы. Соберите jar проекта.

В репозиторий загружены файл docker-compose.yml, Dockerfile, .env. 

Скачайте их, переместите вместе с jar файлом проекта в удобную для Вас директорию и ввердите в командную строку 
docker-compose up --build 
(при условии наличия Docker).
После этого откройте любой браузер и перейдите сюда: http://localhost:8080/swagger-ui/index.html
 
Для авторизации в роли админа предусмотрен заранее созданный пользователь admin (пароль admin).
Если Вас не устраивает параметры пользователя, измените их по своему усмотрению.
